This program assists high school educators who are beginning to study electrostatics. It enables visualization of complex electrical charge systems with reasonable accuracy.

*Version 1.0.2* - Recent improvements:
  - Upgraded numerical integration from Euler's method to fourth-order Runge-Kutta
  - Enhanced optimization and accuracy
  - Eliminated overlapping of field lines with charge surfaces
  - Better adherence to fundamental electrostatic principles
  - Added distance measurement between charges to facilitate Coulomb's Law calculations
  - Implemented rendering capability to save and print visualizations of charge systems
*Plans on future* updates:
  -Adding moving characteristics to charges to able the user to build more complex systems with a possibility of moving charges in space-time. Furthemore      visiualizing lines of magnetic fields produced by those movements
*System requirements/used frameworks*:
	-The program is created using the QT creator which provides a user friendly and optimised interface. But this also has it cons: in order to launch a program user has to download additional libraries, resulting in a slightly greater memory usage.
	-Application is optimised for all downlisted OCs:
			1)windows 98 - windows 11 (x32-x64)
			2)Compatible with recent versions of macOS
